Mais cela est pour les titres de niveau 1 — utilisez <h2> pour un titre de niveau 2, etc.

Maintenant que vous savez comment afficher un titre de tous les niveaux, il faut l'aligner, lui donner une couleur ou une taille adéquate grâce au CSS — un autre langage de mise en page qui marche avec le HTML et qui sert au style. Le meilleur site pour ça serait sans doute Codepen.io :

Donc vous savez mettre un titre, mais pour rédiger un paragraphe, il faut utiliser la balise <p>mon texte de paragraphe</p>. Maintenant, le CSS !

LE CSS 🥳

Le CSS sert donc à mettre de la couleur, aligner, etc.

Voici à quoi ressemble un éditeur CSS 🔥 :

On retrouve bien h1 mais sans <>, car il faut juste écrire (avec la syntaxe ci-dessous) :

/// text-align: center; /// pour aligner au centre, left pour gauche, right pour droite 👍 !

/// color: blue; /// pour mettre la police en bleue 💙 !

/// font-family: Arial; /// police Arial 🎯.

On peut aussi changer la couleur du fond dans le CSS en écrivant body puis :

/// background-color: blue; /// couleur bleue 💙.
