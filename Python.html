Apprendre les bases du python 😃


Le python, c'est quoi 🤯?

Python est un langage de programmation polyvalent et de haut niveau, il permet de développer rapidement des programmes tout en restant simple à comprendre. Pour par exemple progammer des robots etc...


Exemple pour afficher un texte 🔥:

print ("Hello world")

Resultat 🏁:

>>>Hello world

Print --> ca veut dire imprimer en anglais 🤓!

Et il faut mettre les parenthèses puis les guillemets entre le texte que l'on veut afficher 👌!


Faire une BOUCLE 🔄 !

Pour faire une boucle, il va faloir écrire :

for i in range (nombre de répétitions):

puis,par exemple, à la ligne, mettre print ("Hello world") et ça va répéter l'écriture du mot en fonction du nombre que l'on a mis auparavant 🥳!

Faire une question 🐸:

Pour faire une question, il va faloir utiliser "input". Voici un exemple 🔥:

nom = input ("Quel est ton prénom ? ")

print ("Bonjour", nom, "!")


Ce que ça fait 🤩: ca demande le prénom , vous répondez, et le code vous répond "Bonjour (puis votre prénom)!"
